# Tableau_netflix_dashboard_project
Netflix Content Dashboard Project 

This is an interactive dashboard exploring Netflix's content library, featuring analysis of movies vs. TV shows, release trends by country, and top-rated content. It is built with Tableau to provide insights into streaming trends and content strategy.

The dashboard can be viewed at the following link:
https://public.tableau.com/views/NetflixContentdashboard/Netflixdashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link
